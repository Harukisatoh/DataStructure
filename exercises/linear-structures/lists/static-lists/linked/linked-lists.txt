Listas encadeadas são um tipo de lista na qual a ordem lógica dos elementos não é a mesma da ordem física, ou seja, elementos vizinhos não ficam necessariamente em posições vizinhas da memória. Por esse motivo, cada elemento precisa indicar onde está seu sucessor.

Linked lists are a type of list in which the logical order of the elements is not the same as the physical order, that is, neighboring elements do not necessarily lie in neighboring positions of memory. For this reason, each element must indicate where its successor is.
